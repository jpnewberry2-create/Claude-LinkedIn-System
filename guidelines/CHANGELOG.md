## v4.0 — 21/06/2026

Current canonical guideline: `guidelines/brand-guidelines-v4.0.html`.

Visual identity at v4.0, building on the locked v3.5 system:
- Fonts: Fraunces (SemiBold display) + Inter (400–900 body/UI), with the JN-named font files (FrauncesJN, InterJN families, JN-Quote-BigMarks).
- Palette: Canvas, Ink Blue, Dark Blue, Charcoal, Slate, plus Accent Pink for CTA only (under 5%). All text/background pairings meet WCAG 4.5:1.
- Slide system: frosted-glass squircle panel over a baked gradient background; squircle geometry finalised via vector.
- Background-to-pillar mapping: V3 cool (News & Insights), V4 warm-pink (Thought Pieces), V2 pink (Best Practice), V1 warm (reserve).
- Title mark: the name is the brand; no pictorial logo. Footer lockup is pink dot + wordmark + page number.
- Carousel system locked at V7.

Repository structured and brand materials uploaded. Design system (in `/design-system/`) to be derived by Claude Design in a following phase.
